30 November 2014

Ruby is an object oriented programming language, which means that everything is an object. Much like the world around us, we have objects everywhere- tables, trees, televisions and cars are all objects. Even you are an object. All objects in real life have attributes. Think about tables for example. Tables have different materials, lengths, and heights. These are all examples of attributes.
